## Run K6 Test
```
k6 run nest.js
```

or 

```
k6 run go.js
```

</br>

## How to run nest js

Go inside `./nest` directory, then run

```
yarn
```

```
yarn start
```

</br>


## How to run go

Go inside `./go` directory, then run

```
go mod tidy
```

```
go run main.go
```

</br>

## How to run rust

Go inside `./rust` directory, then run

```
cargo r -r
```
